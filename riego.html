¿Cómo funciona?

El sistema utiliza sensores de humedad y un micro:bit programado para regar automáticamente cada 3 días si el suelo está seco.

Ventajas

  • Evita desperdicio de agua.
  • Fácil de usar para personas mayores.
  • Sustentable y adaptable a diferentes cultivos.
